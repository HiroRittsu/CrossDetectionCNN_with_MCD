# Cross detection CNN with MCD
細胞画像の領域検出に適した時系列を考慮するドメイン適応のためのプログラム

# Requirement

* Python 3.8.3
* Anaconda 4.8.3

# Installation

```bash
conda env create --file ./anaconda/env.yml
```

# Usage

```bash
conda activate cross_detection_cnn
python3 main.py
```

# Note
※ 対象となるデータは付属していないため、全てが動くわけではありません。
